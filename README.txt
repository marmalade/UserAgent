Name: Marmalade UserAgent
Description: Provides a method to query Android / iOS device for the browsers user-agent string
Developer: Mat Hopwood
Contact: http://www.pocketeers.co.uk

Usage:

1. Copy the UserAgent folder to the extensions folder in the Marmalade SDK
2. Add UserAgent to the subprojects area of your project MKB file
3. Add UserAgent.h to your source file
4. Call getUserAgent() to retrieve the user-agent
